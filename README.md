# Pharmacy Management System

## Description

A simple Pharmacy Management System built using HTML, CSS, and JavaScript. The application allows users to add and manage medicines by storing their names and prices through an easy-to-use interface.

## Features

* Add new medicines
* Store medicine names and prices
* Display medicine records
* Simple and user-friendly interface
* Dynamic data management using JavaScript

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Open the project in a web browser.
2. Enter the medicine name and price.
3. Add the medicine to the list.
4. View and manage the stored medicines.

## Project Purpose

This project was developed to practice front-end web development using HTML, CSS, and JavaScript while building a simple pharmacy management application.
